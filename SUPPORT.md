# 🌈 SAGE Terminal — Support

Thanks for using **SAGE Terminal**! Here's how to get help.

## 🟣 How to Get Help

| 🎨 Channel | Purpose |
|---|---|
| 🔵 [GitHub Issues](https://github.com/youngsage22/sage-terminal/issues) | Bug reports and feature requests |
| 🟠 [GitHub Discussions](https://github.com/youngsage22/sage-terminal/discussions) | Questions, ideas, and general chat |
| 🟢 [Documentation](./doc/user-docs/) | Guides, settings, and configuration |

## 🔴 Reporting Bugs

When reporting a bug, please include:
- **SAGE Terminal version**
- **Windows version**
- **Steps to reproduce**
- **Expected vs. actual behavior**
- **Screenshots** (especially useful for color/rendering issues!)

## 🟡 Feature Requests

We love new ideas! Open a [GitHub Discussion](https://github.com/youngsage22/sage-terminal/discussions) and label it as a feature request.
