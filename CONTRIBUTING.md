# 🌈 Contributing to SAGE Terminal

Thank you for your interest in making **SAGE Terminal** even better! We welcome contributions of all kinds.

## 🎨 Ways to Contribute

```
🟣 Report bugs
🔵 Suggest new color themes
🟠 Improve documentation  
🟡 Fix issues
🟢 Add new features
🔴 Review pull requests
```

## 🚀 Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/sage-terminal.git`
3. **Create** a branch: `git checkout -b feature/your-colorful-feature`
4. **Make** your changes
5. **Test** your changes thoroughly
6. **Push**: `git push origin feature/your-colorful-feature`
7. **Open** a Pull Request 🎉

## 🎛️ Development Setup

See [doc/building.md](./doc/building.md) for full build instructions.

**Requirements:**
- Visual Studio 2022 (C++ Desktop workload)
- Windows 11 SDK
- .NET Framework 4.8

## 🎨 Adding a Color Theme

Color themes live in the settings JSON. To add a new theme:

1. Create a scheme entry with your colors
2. Name it something vivid and memorable
3. Test it across different shells (PowerShell, CMD, WSL)
4. Submit a PR with screenshots!

## 📋 Pull Request Checklist

- [ ] 🟢 Code builds without errors
- [ ] 🔵 Tests pass
- [ ] 🟠 Documentation updated if needed
- [ ] 🟣 Screenshots included for visual changes
- [ ] 🔴 No breaking changes without discussion
